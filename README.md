# Responsive E-Commerce Website

This project is a responsive e-commerce website with various features that allow users to shop online, search for products, and manage their shopping cart. The website includes multiple sections, such as new arrivals, customer reviews, and a contact form.

## Project Structure

The project includes the following files:

- `index.html`: The main structure of the webpage.
- `style.css`: Styles used for responsive design and layout.
- `main.js`: JavaScript for various website functions (cart, wishlist, search, etc.).
- `img/`: A folder containing product images and other graphical elements for the website.

## Features

### Navbar
The navigation bar includes links to different sections of the website:
- `Home`: The homepage
- `New Arrival`: New product arrivals
- `Products`: The product list
- `Reviews`: Customer reviews

There are also additional icons for search, cart, wishlist, and user login.

### Home Page
- A large hero image with a call-to-action button for purchasing new products.

### New Arrival
- A slider featuring newly added products to the website, created using `swiper.js`.

### Product List
- Users can sort products by price (ascending or descending).
- Ability to add products to the shopping cart and wishlist.

### Cart
- Users can add products to their cart and view the total quantity and price.
- There is also an option to remove products from the cart.

### Wishlist
- Users can add their favorite products to the wishlist.

### Login and Sign Up
- Users can log in and sign up to manage their accounts.

### Customer Reviews
- A section displaying customer reviews with graphical cards and star ratings.

### Contact Us
- A form for users to contact customer support.

### Promotional Modal
- A promotional modal with a special offer appears after 3 seconds on the page.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
- Swiper.js for product sliders
- Boxicons for icons
